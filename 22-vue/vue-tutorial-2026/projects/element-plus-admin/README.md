# Element Plus 后台管理系统项目

状态：规划已建立，待 Batch 07 生成。

## 项目目标

做一个足够应付 Vue3 全栈面试的后台管理系统，覆盖真实项目中最常见的能力：

- 登录页。
- 首页仪表盘。
- 用户管理。
- 角色管理。
- 菜单与权限。
- 表格查询。
- 新增、编辑、删除。
- 路由守卫。
- Pinia 用户状态。
- axios 请求封装。
- Element Plus 表单校验。
- TypeScript 接口类型。

## 技术栈

- Vue3
- Vite
- TypeScript
- Vue Router
- Pinia
- axios
- Element Plus
