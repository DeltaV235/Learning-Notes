## 本章定位

TypeScript 对 Java 后端开发者并不陌生，本质上是给 JavaScript 增加静态类型分析能力。学习重点是能看懂并编写 Vue 项目中的类型，而不是炫技式类型体操。

## 计划内容

1. 六何分析 TypeScript。
2. 安装 TypeScript 环境。
3. HelloWorld 和 `tsconfig.json`。
4. 基础类型、类型推断、联合类型、类型断言。
5. 数组、元组、枚举。
6. 函数、接口、类、泛型。
7. 类型别名、获取类型、内置对象类型、声明文件。
8. 常用操作符：`keyof`、`typeof`、索引访问类型、映射类型。

## 面试重点

- `interface` 和 `type` 的区别。
- 泛型解决什么问题？
- 联合类型和类型收窄如何使用？
- 为什么 Vue + TS 项目通常需要 `vue-tsc`？
