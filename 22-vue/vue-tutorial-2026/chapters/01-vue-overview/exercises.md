# 第 1 章练习

## 基础题

1. 用自己的话解释 Vue 的“声明式渲染”。
2. 说明 Vue Router、Pinia、axios 分别负责什么。
3. 写出创建 Vue3 项目的命令。
4. 解释为什么新项目优先选择 Vue3 而不是 Vue2。

## 思考题

1. Java 后端里的 Controller、Service、DTO，分别可以和前端项目里的哪些概念类比？
2. 后台管理系统里，哪些状态适合放在组件内部，哪些适合放在 Pinia？

## 实操题

1. 安装 Vue - Official 扩展。
2. 在浏览器中安装 Vue Devtools。
3. 运行本项目根目录的索引页，观察页面组件结构。
