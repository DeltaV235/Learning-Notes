## 本章定位

Vue Router 是后台管理系统的页面组织核心。Java 后端开发者可以把它理解为前端 SPA 内部的路由层，但它不等同于 Spring MVC 的服务端路由。

5. 嵌套路由。
6. 动态路由传参：`params` 和 `query`。
