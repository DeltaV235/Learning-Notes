# 第 8 章 UI 框架

状态：目录已建立，待 Batch 05 生成正文、示例和练习。

## 本章定位

本课程以 Element Plus 为后台管理系统重点，Vant4 作为移动端 UI 框架了解。

## 计划内容

1. 功能性框架与 UI 框架的配合。
2. UI 框架分类与常用组件。
3. Element Plus 完整引入。
4. Element Plus 按需引入。
5. 后台业务页面设计：表格、表单、分页、弹窗、消息提示。
6. Vant4 完整引入和按需引入。
7. 移动端业务页面了解。

## 面试重点

- UI 框架解决什么问题，不能解决什么问题？
- Element Plus 表单校验如何组织？
- 表格查询、分页、删除确认如何组合？
- 为什么后台系统不建议过度自定义 UI？
